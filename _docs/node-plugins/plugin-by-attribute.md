Drag the rectangle. The Node is made draggable by defining the attribute *plugin-dd="true"*.

```html
<body>

</body>
```

```js
<script src="itsabuild-min.js"></script>
<script>
    document.getElement('.container').setAttr('plugin-dd', 'true');
</script>
```
