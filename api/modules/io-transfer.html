io-transfer Module

Defined in: src/io/extra/io-transfer.js:3

Available since 0.0.1

Extends io by adding the method readXML to it. Should be called using the provided mergeInto-method like this:

Example:

var IO = require("io"); var IOtransfer = require("io-transfer"); IOtransfer.mergeInto(IO);
